gh-contribution-planner (dots-ai distribution)
==============================================

This skill is authored by dots-ai S.A. for distribution via the dots-ai
workstation. It is released under the Apache License 2.0 (see LICENSE.txt).

The skill orchestrates work that delegates to peer dots-ai skills:

  - github-cli-workflow — push + draft PR creation
  - gh-fix-ci          — red GitHub Actions triage
  - gh-address-comments — reviewer comment resolution
  - dev-assistant      — pre-edit repository discovery

`scripts/inspect_contributions.py` is original work and contains no
third-party code. Stdlib only; relies on the `gh` CLI for GitHub access.
