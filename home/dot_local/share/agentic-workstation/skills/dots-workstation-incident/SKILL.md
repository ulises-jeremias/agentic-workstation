# Incident (WHAT)

Use for unplanned events that disrupt or degrade production systems, live users, or critical operations.

## Default guardrails

1. Apply **`dots-workstation-output-handshake`** before final output.
2. Capture immediate mitigation separately from root cause and follow-up actions.
3. Use **`dots-workstation-meeting-minutes`** if a validation or post-incident meeting needs minutes.
4. Create/link follow-up bugs or tasks only after approval.
